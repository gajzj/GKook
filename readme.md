## 基本命令格式
- ****所有的命令以 `/` 开头，如 `/hello`
- 如果该命令需要传递参数，参数和用小括号括住 `/command(arg1,arg2,arg3)`
- 多级命令通过 `.` 执行二级命令 `/command1(arg1).command2`
